Commonly used Endpoints
Shutdown: POST - http://localhost:9285/__admin/shutdown
Ping: GET - http://localhost:9285/ping
Reset: GET - http://localhost:9285/__admin/mappings/reset

You can start the wiremock server using the start.bat file if you are using a windows machine or use the same commands in it if you are using a non windows machine.
